# fmi-probability-and-statistics-2024-2025
Solutions in R for problems during the mandatory course Probability and Statistics 2024-2025 in FMI, Sofia University.  

# Contents
- directories:
  - data - the data used for the problems
  - solutions - R files with solutions for the problems
- problems - file that contains the formulations of the problems
- notes - notes on R and statistics, used in the course

# Problems
- Problems 1–17: Estimating probabilities using simulation methods
- Problems 18–31: Computing probabilities using built-in functions for discrete distributions
- Problems 32–48: Simulation and Analysis of Continuous Probability Distributions
- Problems 49–59: Exploratory Data Analysis of Real Datasets
- Problems 60–69: Central Limit Theorem and Normal Approximation
- Problems 70–79: Statistical Hypothesis Testing
- Problems 80–88: Hypothesis Testing for Two Samples
- Problems 89–97: Confidence Intervals
- Problems 98–106: Chi-Square Tests and Categorical Data Analysis
- Problems 107–114: Regression Analysis and Modeling
- Problems 115-123: Review of Statistical Methods
